How to run this online for free

Best option: Streamlit Community Cloud
1. Create a free GitHub account.
2. Create a new GitHub repository.
3. Upload all files from this folder into that repository.
4. Go to Streamlit Community Cloud and create a new app.
5. Choose the repository and set the main file as app.py.
6. Deploy.

Alternative: Replit
1. Create a Python Repl.
2. Upload all files.
3. In the Shell, run:
   pip install -r requirements.txt
   streamlit run app.py --server.port 3000 --server.address 0.0.0.0

Files
- app.py: the dashboard application
- area_master.csv: editable area, price, CAGR, water, infra, and economic thesis data
- development_sources.csv: development tracker with source URLs
- requirements.txt: packages needed by Streamlit Cloud/Replit

Important caution
Prices are directional listing-market indicators. Before purchase, validate: sale deed chain, conversion order, RERA/applicable approval, planning authority approval, e-khata/A-khata status, road width, water source, drainage/flood exposure, encumbrance certificate, litigation, and actual recent transaction comparables.
